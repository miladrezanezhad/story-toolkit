Quick Start

Build your first story with Story Development Toolkit v2.0.0 in 5 minutes

NEW in v2.0.0: Optional LLM support for advanced dialogue generation with OpenAI, Anthropic, and local models!

Step 1: Installation

bash 
git clone https://github.com/miladrezanezhad/story-toolkit.git
cd story-toolkit
pip install -r requirements.txt

# NEW in v2.0.0 - Install with LLM support
pip install story-toolkit[openai]      # OpenAI support (GPT-4, GPT-3.5)
pip install story-toolkit[anthropic]   # Anthropic support (Claude)
pip install story-toolkit[local]       # Local LLM support (Ollama, free)
pip install story-toolkit[all]         # All backends

Or install directly with pip install -e .

Step 2: Import the Library

python 
from story_toolkit import StoryToolkit

# Create a Toolkit instance
toolkit = StoryToolkit()

The StoryToolkit class provides unified access to all features.

NEW in v2.0.0 - Enable AI-Powered Features: 
from story_toolkit.llm import LLMFactory, LLMProvider

# Create Mock backend for testing (no API key required)
llm = LLMFactory.create_backend(provider=LLMProvider.MOCK)

# Create Toolkit with LLM capabilities
toolkit = StoryToolkit(llm_backend=llm)

Step 3: Create a Story

python 
# Create a fantasy story with the theme of courage
story = toolkit.create_story(
    genre="fantasy",
    theme="courage",
    complexity=4
)

print(story["metadata"]["genre"])      # "fantasy"
print(story["metadata"]["theme"])      # "courage"
print(story["metadata"]["complexity"]) # 4
Supported genres: fantasy, mystery, romance, adventure, sci_fi

Step 4: Add Characters

python 
# Add the hero
hero = toolkit.add_character_to_story(
    story,
    "Kai",
    "protagonist"
)

# Develop the hero's personality
hero.add_trait("brave")
hero.add_trait("determined")
hero.add_trait("compassionate")
hero.add_goal("Save the kingdom")
hero.add_skill("sword_mastery")
hero.add_skill("stealth")
hero.add_weakness("overconfidence")
hero.add_fear("failure")

# Add the villain
villain = toolkit.add_character_to_story(
    story,
    "Morgan",
    "antagonist"
)
villain.add_trait("manipulative")
villain.add_trait("charismatic")
villain.add_goal("Control the kingdom")
villain.add_skill("strategic_planning")

# Add a mentor
mentor = toolkit.add_character_to_story(
    story,
    "Master Yuki",
    "mentor"
)
mentor.add_trait("wise")
mentor.add_trait("mysterious")

# Create relationships
hero.add_relationship("Morgan", "enemy", strength=9)
hero.add_relationship("Master Yuki", "mentor", strength=10)
Available roles: protagonist, antagonist, supporting, mentor

Step 5: Generate Dialogue (Basic Mode)

python 
# Generate a conflict dialogue
dialogue = toolkit.dialogue_gen.generate_dialogue(
    "Kai",
    "Morgan",
    context="conflict",
    mood="tense"
)

for line in dialogue:
    print(line)
Output:

Kai: I can't believe you would do this!
Morgan: You left me no choice.
Kai: There's always a choice. You just chose wrong.

Step 5-A: Generate Advanced Dialogue with AI NEW in v2.0.0

python 
from story_toolkit.llm import LLMFactory, LLMProvider

# Step 1: Create LLM backend (Mock for testing, no API key)
llm = LLMFactory.create_backend(provider=LLMProvider.MOCK)

# Step 2: Create Toolkit with LLM capabilities
toolkit_advanced = StoryToolkit(llm_backend=llm)

# Step 3: Generate advanced dialogue
advanced_dialogue = toolkit_advanced.dialogue_gen.generate_dialogue(
    "Kai",
    "Morgan",
    context="final_battle",
    use_advanced=True,      # Enable AI
    style="dramatic",         # Options: natural, dramatic, poetic, humorous
    num_lines=8
)

for line in advanced_dialogue:
    print(line)
Output (Mock backend example):

Kai: I can't believe what you've done!
Morgan: You left me no choice, Kai.
Kai: There's always a choice. You chose wrong.
Morgan: We'll see who was wrong in the end.
Kai: This isn't over.
Morgan: You're right about that.
Kai: Next time, you won't be so lucky.
Morgan: We'll see about that.
Note: For real AI (OpenAI or Claude), you'll need an API key. The MOCK backend is perfect for testing without any API key.
Available styles: natural, dramatic, poetic, humorous

Step 6: Build a World

python 
from story_toolkit.core.world_builder import WorldBuilder

world = WorldBuilder()
world.create_world("Eldoria", "fantasy")

# Add locations
world.add_location("Crystal City", "Ancient metropolis", "city")
world.add_location("Shadow Forest", "Mysterious dark forest", "forest")
world.add_location("Thunder Mountains", "Storm-peaked mountains", "mountains")

# Connect locations
world.connect_locations("Crystal City", "Shadow Forest")

# Add world rules
world.add_rule("magical", "Only eclipse-born can wield magic")
world.add_rule("social", "The Council of Seven rules all")

# Add a faction
world.add_faction(
    "Shadow Guild",
    "Secret underground organization",
    goals=["Control magic", "Overthrow council"]
)

Step 7: Check Story Coherence

python 
# Check story coherence and get a report
report = toolkit.check_story_coherence(story)

# Display overall score
print(f"Coherence Score: {report['overall_score']:.0%}")

# Check for plot holes
if report['plot_holes']:
    print("\n⚠️ Plot Holes Found:")
    for hole in report['plot_holes']:
        print(f"  - {hole}")

# Show recommendations
if report['recommendations']:
    print("\n💡 Recommendations:")
    for rec in report['recommendations']:
        print(f"  - {rec}")
Output:

Coherence Score: 100%

💡 Recommendations:
- Story appears coherent - maintain this quality throughout!

Step 8: Save & Export

python 
from story_toolkit.utils.helpers import save_story, export_to_markdown

# Save as JSON
save_story(story, "my_story.json")

# Export as Markdown (great for GitHub)
export_to_markdown(story, "my_story.md")

# Or use the Toolkit's built-in export
toolkit.export_story(story, format="json", filename="my_story")
toolkit.export_story(story, format="markdown", filename="my_story")

print("✅ Story saved successfully!")

🎯 Complete Code (Basic)

python 
from story_toolkit import StoryToolkit
from story_toolkit.utils.helpers import save_story

# 1. Create instance
toolkit = StoryToolkit()

# 2. Create story
story = toolkit.create_story("fantasy", "courage", complexity=4)

# 3. Add characters
hero = toolkit.add_character_to_story(story, "Kai", "protagonist")
hero.add_trait("brave")
hero.add_goal("Save the kingdom")

villain = toolkit.add_character_to_story(story, "Morgan", "antagonist")
villain.add_trait("manipulative")

# 4. Create relationship
hero.add_relationship("Morgan", "enemy", strength=9)

# 5. Generate dialogue
dialogue = toolkit.dialogue_gen.generate_dialogue(
    "Kai", "Morgan", context="conflict"
)
for line in dialogue:
    print(line)

# 6. Check coherence
report = toolkit.check_story_coherence(story)
print(f"\n✅ Coherence Score: {report['overall_score']:.0%}")

# 7. Save story
save_story(story, "my_story.json")
print("📁 Story saved to my_story.json")
